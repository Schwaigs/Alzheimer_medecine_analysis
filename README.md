# Alzheimer medicine analysis

Ce dépôt contient un projet réalisé dans la cadre du cours de probabilités et statistiques que j'ai suivie à l'IUT Robert Schuman. Le but était d'analyser la phase de test d'un médicament contre la maladie d'Alzheimer. Il s'agissait d'étudier l'efficacité du médicament en comparant les données recueillies sur 3 groupes de patients. 2 d'entre eux ayant des doses différentes du médicament et le 3eme ayant reçu un placebo.

Dieses Repository enthält ein Projekt, das im Rahmen des Wahrscheinlichkeits- und Statistikkurses, den ich in der IUT Robert Schuman belegen habe realisiert wurde. Das Ziel war die Testphase eines Medikaments gegen die Alzheimer-Krankheit zu analysieren. Dazu wird durch die vergleichung von die Daten von 3 Patientengruppen die Wirksamkeit des Medikaments untersuchen. Zwei von ihnen bekamen unterschiedliche Dosen des Medikaments und der dritte erhielt ein Placebo.

This repository contains a project realized for the probability and statistics course that I followed at the IUT Robert Schuman. The purpose was to analyse the test phase of a medicine against Alzheimer's disease. The aim was to study the efficacy of the medicine by comparing the data collected on 3 groups of patients. Two of them had different doses of the medicine and the third one received a placebo.
